# Contributing

## Getting Started

### Prerequisites

- Python 3

### Setup

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd mq-demo
   ```

2. Create and activate a virtual environment:

   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

3. Install dependencies:

   ```bash
   pip install pytest
   ```

4. Run the test suite:

   ```bash
   pytest -v
   ```
