# Contributing

## Getting Started

### Clone the repository

```bash
git clone <repository-url>
cd mq-demo
```

### Create a virtual environment

```bash
python3.12 -m venv .venv
source .venv/bin/activate
```

### Install dependencies

```bash
pip install pytest flake8
```

### Run the test suite

```bash
pytest
```

### Run the linter

```bash
flake8
```
