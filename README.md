## Test Setup

- Create and activate a Python virtual environment: `python3 -m venv venv && source venv/bin/activate`
- Install the test dependency: `pip install pytest`
- Run the tests: `pytest test_math_operations.py -v`

## Troubleshooting

- `pytest: command not found`: ensure the virtual environment is activated via `source venv/bin/activate`
- `ModuleNotFoundError: No module named 'pytest'`: run `pip install pytest` inside the activated virtual environment
